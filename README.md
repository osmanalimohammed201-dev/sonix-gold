# Sonix Gold 🎧

Sonix Gold is a premium interactive 3D headphone showcase inspired by modern e-commerce experiences like Amazon 360° product viewers.

## Features

- 🎧 Interactive 360° Product Viewer
- 🎨 Dynamic Headphone Color Customization
- 🌈 Auto-Changing Background Themes
- ✨ Smooth Animations & Transitions
- 📱 Fully Responsive Design
- ⚡ Fast Performance
- 🔄 Auto-Rotation & Zoom Controls
- 🌟 Premium UI/UX Experience

## Tech Stack

- React
- Three.js
- React Three Fiber
- Tailwind CSS
- JavaScript
